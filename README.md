# Virtual Events Utility Scripts

This is a set of PHP scripts to import data (sessions, speakers) to Kaltura.

# Setup and Running

1. Clone this repo.
2. `composer install` 
3. Copy config.template.php into config.php and adjust the settings as instructed in the inline comments
4. Use each respective script according to your need.

# Important Notes

1. This script assumes PHP 7 and above.
